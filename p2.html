Project #2 Face Generator

Design Process

I started with following my sketch to create a still face. After finishing the face I then started to add variables. The interesting part of this process is that I let different codes create "movement", which I didn't decide in the first place. I try creating different sizes of random to make the eyes movement. and use mouse position to change the different sizes of the face. Filing the color is my last step cause I try to keep my code clear and simple while I testing the variables.
