#8 Experimental Camera

Description

Design Process

I play with different DOM, interface elements and end up choosing to create a button that can change the background color. In my previous attempt, the background color is locked in order to create a pop-art feeling but I think the randomness, make my camera filter more interactable and, more fun.

Reflection

With the interface elements, I think it's a great chance to make the viewer feel more interactive with the work. Viewers can choose and recreate existing images based on their personal preferences under limited constraints. I think this is a very practical technique, and I personally like it lot-the interactive buttons make the work more personal and at the same time, unique
